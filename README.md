# 🌿 Mundo das Plantas — Jogo Educativo de Botânica

Jogo digital interativo para alunos do **7º ano do Ensino Fundamental**, cobrindo o conteúdo de botânica de Ciências da Natureza.

---

## 📚 O que tem no jogo?

| Atividade | O que o aluno faz |
|---|---|
| 🔍 Explorar as partes | Toca em cada parte da planta e lê a explicação |
| 🧩 Monte a planta | Arrasta os nomes para o lugar certo |
| 🔄 Ciclo de vida | Descobre cada etapa, da semente ao fruto |
| ☀️ Fotossíntese | Mexe nos controles de luz, água e CO₂ e vê o efeito |
| 🔤 Adivinhe a palavra | Lê a dica e digita a resposta |
| ⭐ Quiz final | Responde 6 perguntas e recebe pontuação |

## 🚀 Como disponibilizar para os alunos

### Opção 1 — Netlify Drop (recomendado, gratuito, 2 minutos)

1. Acesse **netlify.com/drop** no navegador
2. Arraste a pasta `botanica-app` para a página
3. Pronto! O site vai gerar um link como `https://nome-aleatorio.netlify.app`
4. Compartilhe esse link com os alunos pelo WhatsApp ou grupo da turma

### Opção 2 — Pen drive ou rede local

1. Copie a pasta `botanica-app` para um pen drive
2. Abra o arquivo `index.html` diretamente no navegador do computador da escola
3. O jogo funciona sem internet!

### Opção 3 — GitHub Pages (gratuito, permanente)

1. Crie uma conta gratuita em **github.com**
2. Crie um repositório novo e faça upload dos arquivos
3. Vá em Settings → Pages → selecione a branch `main`
4. O link fica disponível para sempre

---

## 📲 Como instalar no celular dos alunos

### Android
1. Abra o link do jogo no **Google Chrome**
2. Toque nos **3 pontinhos** no canto superior direito
3. Toque em **"Adicionar à tela inicial"**
4. O ícone 🌿 vai aparecer como um app normal

### iPhone / iPad
1. Abra o link no **Safari**
2. Toque no botão de **compartilhar** (quadrado com seta para cima)
3. Toque em **"Adicionar à Tela de Início"**
4. O ícone 🌿 vai aparecer na tela inicial

> 💡 Depois de instalado, o jogo funciona **sem internet**!

## ✅ Conteúdos abordados

- Partes da planta: raiz, caule, folha, flor, fruto e semente
- Funções de cada parte
- Fotossíntese: ingredientes, processo e produtos
- Cloroplasto, clorofila e estômatos
- Ciclo de vida das plantas
- Germinação e reprodução
- Vocabulário: xilema, floema, glicose, seiva
